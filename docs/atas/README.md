# ATAs de Acompanhamento Individual

Esta pasta contém as Atas de Acompanhamento Individual de cada membro da equipe.

## Estrutura das ATAs

Cada ATA contém um **Registro de Atividades Semanais** com 18 semanas:

| Coluna | Descrição |
|--------|-----------|
| **Período** | Data de início e fim da semana (formato DD/MM - DD/MM) |
| **Microfundamento** | Conteúdo/tema a ser estudado na semana |
| **Estudou?** | Check para o aluno confirmar se estudou o microfundamento (✅ ou ⬜) |
| **Atividade Prevista** | O que foi planejado para a semana |
| **Atividade Realizada** | O que foi efetivamente realizado |

**Legenda da coluna Estudou?:**
- ✅ = Estudou o microfundamento
- ⬜ = Não estudou ainda

## Como Preencher

### Para o Professor

1. Preencha a coluna **Microfundamento** no início do semestre com os temas de cada semana
2. Atualize o nome do aluno no título de cada arquivo

### Para os Alunos

1. Ao estudar o microfundamento da semana, altere a coluna **Estudou?** de ⬜ para ✅
2. No início de cada semana, preencha a coluna **Atividade Prevista** com o que você planeja fazer
3. No final de cada semana, preencha a coluna **Atividade Realizada** com o que você efetivamente fez
4. Seja específico e detalhado nas descrições
